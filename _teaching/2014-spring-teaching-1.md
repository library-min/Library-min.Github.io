학부생을 대상으로 한 HTML, CSS, JavaScript 기초 강의입니다.

강의 내용
======
- HTML 기본 구조와 의미적 마크업
- CSS 레이아웃과 스타일링
- JavaScript 기본 문법과 DOM 조작

실습 프로젝트
======
- 개인 웹사이트 제작
- 대화형 웹 애플리케이션 개발

평가 및 피드백
======
학생들의 학습 만족도가 높았고, 실무에 직접 적용할 수 있는 내용으로 구성되어 좋은 평가를 받았습니다.
